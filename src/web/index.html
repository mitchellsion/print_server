How to import on each OS
  • iOS/iPadOS: AirDrop or email the .crt to the device, open it, install the profile in Settings → General → VPN & Device Management, then enable it in Settings → General → About → Certificate Trust Settings.
  • Android: Settings → Security → Encryption & credentials → Install a certificate → CA certificate, pick the downloaded .crt.
  • macOS: double-click the .pem, add to login keychain, then in Keychain Access set the cert's trust for "Secure Sockets Layer" to "Always Trust".
  • Windows: double-click the .crt → Install Certificate → Current User → Place all certificates in the following store → Trusted Root Certification Authorities.
  • Linux (Chrome/Chromium): certutil -d sql:~/.pki/nssdb -A -t P,, -n print-server -i print-server.pem (requires libnss3-tools).

Reach the server by a name or IP that appears in the SAN list above — otherwise TLS hostname verification will fail even with the cert trusted.
